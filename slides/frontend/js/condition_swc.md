# JavaScript
Les conditions

<kbd>switch/case</kbd>
```js
let b = 'bleu'
switch (b) {
  case 'bleu':
    afficherlheure()
    break
  case 'vert':
    console.log("code éxécuté si b égale 'vert'")
    break
  default:
    // Instructions à exécuter lorsqu'aucune des valeurs ne correspond
    break
}
```
