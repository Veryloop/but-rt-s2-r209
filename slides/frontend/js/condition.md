# JavaScript
Les conditions

<kbd>if/else</kbd>
```js
if (x > 50){
  // faire quelque chose
} else if (x > 5) {
  // faire autre chose
} else {
  // faire encore autre chose
}
```

<kbd>condition terneaire</kbd>
```js
// condition ? <expression si vrai> : <expression si faux>
age = 19
"Je suis " + (age >= 18 ? "majeur" : "mineur")
```
