# PHP
Condition : switch

L'instruction <kbd>switch</kbd> équivaut à une série d'instructions <kbd>if</kbd>. 
```php
<?php
switch ($i) {
    case 0:
        echo "i égal 0";
        break;
    case 1:
        echo "i égal 1";
        break;
    case 2:
        echo "i égal 2";
        break;
}
?>
```
